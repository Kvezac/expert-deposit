## Deposit calculate
***
### Create a new directory:
```bash
mkdir deposit
```
***
### Change the directory:
```bash
cd deposit
```
***
### Git clone:
```bash
git clone git@github.com:Kvezac/expert-deposit.git
```
***
### Create docker image:
```bash
docker compose -d build
```
***
### Start docker:
```bash
docker compose run
```
***
### [Open localhost](http://127.0.0.1:8000/api/docs)
***
